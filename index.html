Método de Bisección

Método cerrado que divide el intervalo a la mitad. Garantiza convergencia si la función es continua y hay cambio de signo.

\[x_r = \frac{x_a + x_b}{2}\]

Falsa Posición

Mejora la bisección uniendo los puntos extremos con una línea recta para aproximar la raíz por interpolación lineal.

\[x_r = x_b - \frac{f(x_b)(x_a - x_b)}{f(x_a) - f(x_b)}\]

Punto Fijo

Transforma \(f(x)=0\) en \(x=g(x)\). El método converge si \(|g'(x)| < 1\) en el entorno de la raíz.

\[x_{i+1} = g(x_i)\]

Newton-Raphson

Utiliza la tangente de la función. Es el método más eficiente (convergencia cuadrática) pero requiere conocer la derivada.

\[x_{i+1} = x_i - \frac{f(x_i)}{f'(x_i)}\]

Método de la Secante

Similar a Newton, pero sustituye la derivada por una aproximación basada en dos puntos anteriores.

\[x_{i+1} = x_i - \frac{f(x_i)(x_{i-1} - x_i)}{f(x_{i-1}) - f(x_i)}\]

Método de Müller

Utiliza una parábola (polinomio de segundo grado) que pasa por tres puntos. Permite encontrar raíces complejas.

\[x_3 = x_2 + \frac{-2c}{b \pm \sqrt{b^2 - 4ac}}\]

Método de Bairstow

Específico para polinomios. Encuentra factores cuadráticos del tipo \(x^2 - rx - s\) para hallar raíces reales y complejas.

\[r = r + \Delta r, \quad s = s + \Delta s\]

Regla de Horner

Algoritmo para evaluar polinomios de forma eficiente reduciendo el número de multiplicaciones.

\[b_i = a_i + b_{i+1}x\]

Newton para Sistemas

Extensión para resolver múltiples ecuaciones no lineales simultáneamente usando la matriz Jacobiana.

\[[J]\{\Delta x\} = -\{f\}\]
